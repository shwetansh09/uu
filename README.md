## 🚀 How to Deploy on GitHub Pages (Step by Step)

### Step 1 — Create a GitHub Account
Go to [github.com](https://github.com) and sign up if you don't have an account.

### Step 2 — Create a New Repository
1. Click **+** (top right) → **New repository**
2. Name it e.g. `sks-grocery`
3. Set to **Public**
4. Click **Create repository**

### Step 3 — Upload ONLY These 2 Files
1. Click **uploading an existing file**
2. Drag and drop:
   - `index.html`
   - `style.css`
3. Click **Commit changes**

> ✅ No images folder needed — all images are built into index.html!

### Step 4 — Enable GitHub Pages
1. Go to your repo → **Settings** tab
2. Left sidebar → **Pages**
3. Under **Source** → **Deploy from a branch**
4. Branch: **main** → Folder: **/ (root)**
5. Click **Save**

### Step 5 — Your Site is Live! 🎉
Wait 1–2 minutes, then visit:
`https://YOUR-USERNAME.github.io/sks-grocery/`
